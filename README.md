# 🎭 El gran teatro del mundo

## Proyecto web educativo sobre Pedro Calderón de la Barca

Página web interactiva dedicada al estudio de la obra **"El gran teatro del mundo"**, uno de los autos sacramentales más importantes del **Siglo de Oro español**.

El proyecto combina literatura, historia y desarrollo web para crear una experiencia educativa moderna y visualmente atractiva.

---

# 📖 Descripción del proyecto

Esta página presenta información sobre:

- El contexto histórico del Siglo de Oro español.
- La vida y obra de Pedro Calderón de la Barca.
- El análisis literario de *El gran teatro del mundo*.
- Sus personajes principales.
- Los valores y temas de la obra.
- Su importancia dentro del teatro barroco.

Además incluye elementos interactivos para mejorar la experiencia del usuario.
